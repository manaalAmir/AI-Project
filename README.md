## How to Run the Project

### 1. Install Required Libraries

```bash
pip install pandas numpy matplotlib scikit-learn
```

### 2. Download the Dataset

Make sure the dataset file (`network_traffic.csv`) is located in the same directory as the notebook.

### 3. Open the Jupyter Notebook

```bash
jupyter notebook AI_Semester_Project.ipynb
```

### 4. Run the Notebook

Open `AI_Semester_Project.ipynb` 
```

The notebook will automatically:

- Load and explore the dataset
- Build a Simple Reflex Agent
- Train KNN, Naïve Bayes, and Logistic Regression models
- Perform K-Means clustering
- Apply Genetic Algorithm feature selection
- Display evaluation metrics and visualizations

### Expected Output

The project generates:

- Accuracy, Precision, Recall, and F1-Score
- Confusion Matrices
- Class Distribution Graphs
- Feature Distribution Histograms
- PCA Visualization for K-Means Clustering
